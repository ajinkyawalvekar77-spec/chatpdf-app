# ChatPDF AI

ChatPDF AI allows users to upload a PDF and ask questions about the document using AI.

## Live Demo
https://chatpdf-app-enter.streamlit.app/

## Features
- Upload PDF documents
- Ask questions about the document
- AI-powered answers
- Fast semantic search

## Tech Stack
- Python
- Streamlit
- FAISS
- Sentence Transformers
- Groq LLM
